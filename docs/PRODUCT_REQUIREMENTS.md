# Требования к продукту (MVP) MustHave

## MVP-цель
Запустить первую версию магазина, где пользователь может:
1. Зайти на сайт.
2. Просмотреть каталог.
3. Открыть карточку товара.
4. Добавить товары в корзину.
5. Оформить заказ (имя, телефон, город/адрес, комментарий).
6. Получить подтверждение, а админ — заявку.

## Основные страницы
- Главная:
  - Баннер.
  - Популярные товары.
  - Короткий блок про бонусы/рефералку (без сложной логики на старте).
- Каталог:
  - Список товаров с ценой, фильтр по простым параметрам (позже).
- Карточка товара:
  - Название, фото, описание, объём, цена.
  - Кнопка «Добавить в корзину».
- Корзина:
  - Список выбранных товаров, итоги по сумме.
  - Кнопка «Оформить заказ».
- Оформление заказа:
  - Поля: имя, телефон, город/адрес, комментарий.
  - (Потом можно добавить: способ доставки, промокод).
- Страница «Спасибо за заказ».

## Админка (простая)
- Авторизация админа (позже можно сделать нормальную RBAC).
- Список товаров.
- Создание/редактирование/деактивация товара.
- Просмотр заявок/заказов.

## Важно
- Оплату онлайн (Kaspi и т.д.) добавляем позже.
- Реальная реферальная логика тоже может быть добавлена вторым этапом, но поля в модели пользователей закладываем сразу.
