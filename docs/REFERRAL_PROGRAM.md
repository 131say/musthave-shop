# Реферальная программа MustHave (черновик)

## Цель
Сделать простую, понятную для людей систему, по ощущениям похожую на MLM, но:
- без сложных матриц и «зазубривания» презентаций,
- с акцентом на реальные покупки и пользу.

## Базовая идея (MVP)
1. У каждого пользователя есть **реферальный код**.
2. Новый клиент может:
   - ввести реферальный код при регистрации/первом заказе,
   - перейти по реферальной ссылке.
3. За это:
   - новому клиенту: приветственный бонус (например, X баллов).
   - пригласившему: бонус после первого оплаченного заказа приглашённого.

## Что НЕ делаем на старте
- Многоуровневый маркетинг с 5–10 уровнями.
- Сложные проценты, ранги и т.п.

## Что ЗАЛОЖИМ в модель заранее
- Возможность связать User → ReferredByUserId.
- Возможность учитывать бонусный баланс (bonusBalance).
- Возможность сохранять события начисления/списания (ReferralEvent).

## Потенциальные сценарии будущего
1. **1-й уровень**:
   - Пользователь получает % или фиксированный бонус с заказов лично приглашенных.
2. **2-й уровень**:
   - Маленький % с «друзей друзей».
3. **Бонус за активность**:
   - При достижении определенного оборота по сети — доп. бонусы.

Но все эти сценарии внедряем ПОЭТАПНО, после запуска простого магазина.
